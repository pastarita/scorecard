## üé® How to Use These Prompts

### For Image Generation Models

1. **Select a prompt** from the relevant markdown file
2. **Copy the prompt** exactly or adapt for your model
3. **Add model-specific parameters** (e.g., aspect ratio, style modifiers)
4. **Generate and iterate** based on results

### Prompt Structure

Each prompt includes:
- **Primary description**: Core visual elements
- **Style modifiers**: Aesthetic direction
- **Color specifications**: Exact hex codes or color descriptions
- **Technical details**: Dimensions, format, composition
- **Context**: Where/how it will be used

### Iteration Strategy

- Start with base prompts
- Refine based on initial results
- Combine elements from multiple prompts
- Experiment with style variations
